# rasp-monitor-bot

Small Telegram Bot to monitor my Raspberry Pi

## Usage

```
  $ export TELEGRAM_TOKEN="YOUR_TELEGRAM_TOKEN"
  $ export OWNER_USERNAME="YOUR_USERNAME"
  $ lein run
```

## REPL

```
  $ lein repl

  (defn my-service [] (-main))


  (my-service)
```
